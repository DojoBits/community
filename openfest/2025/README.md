# OpenFest 2025

![](./SSH-ZTNA-full-latest.jpg)

This is our tech talk about Zero Trust Network Access (ZTNA), where we discuss
the implementation of short-lived certificates, eliminating the need for VPNs
and keys. The talk covers the principles of ZTNA, the benefits of using
short-lived certificates, and how this approach enhances security while
simplifying access management.

Video for the lecture is: [OpenFest2025 - Zero Trust Network Access - Valentin Hristev](https://www.youtube.com/watch?v=JFw1xa6U2sk&t=195s)
